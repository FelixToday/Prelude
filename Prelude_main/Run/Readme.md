# 脚本说明

## **const.py**:
  定义了项目中使用的常量配置，包括各数据集的选项字典（如标签数、最大加载时间）、默认机器与文件基础路径映射等，用于统一管理静态配置。

## **main.py**:

  提供了各种分类模型（除Holmes）的标准训练流程。

## **test.py**:

  模型的推理评估脚本（按加载比例的流量进行评估）。















## **early_main.py**:

  实现动态门控机制流式流量指纹分类的主脚本。涵盖了辅助门控网络的训练、验证评估以及跨多个阈值在真实数据流模拟中的延迟和准确率测试。



## **test_early.py**:
  针对早期检测任务的评估测试脚本。它加载不同基线模型，依据多个置信度阈值在流式到达的数据包上进行实时推理，输出准确率、延迟和流量加载比例等指标。

## **test_module.py**:

  存放实验性质的新型网络模块测试代码。主要包含用于自适应流量对齐的频域操作（如基于随机采样的滚动对齐机制）和基于显著性的序列切片器模块。



## **utils_dataset_metric.py**:
  核心的数据集与评估工具库。负责流量数据的加载、填充截断，初始化各种基线模型的数据加载器，并提供了 KNN 监控和四级分类指标的计算方法。

## **utils_porcess.py**:
  基础的数据处理和配置工具辅助模块。主要用于命令行参数解析、配置文件读取、系统运行配置的自适应调整，以及统一定义了一轮验证评测的具体逻辑。
